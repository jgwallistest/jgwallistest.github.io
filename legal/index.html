How the site is built

There’s no complicated content management system behind this site, so it loads quickly and is very secure. It is built ‘statically’ using the Hugo static site generator (© The Hugo Authors), which is available under version 2.0 of the Apache Licence, and UIkit (© YOOtheme GmbH), which is available under the MIT licence.

Software used to build the site

The different software items used to build this site are:
